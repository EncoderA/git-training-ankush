About This Project

This project demonstrates a practical Git and GitHub workflow.

Project Goals

  • Create small, meaningful commits.
  • Use feature branches for isolated work.
  • Merge changes through GitHub pull requests.
  • Resolve a real merge conflict by editing the conflicted file.

Workflow Note

This page was maintained on the feature/about-page branch before being reviewed through a pull request.

Branch Naming

Feature work uses branches such as feature/about-page and feature/contact-page.
